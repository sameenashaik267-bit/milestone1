--Project Title--

TextMorph – Secure User Authentication System

---Description

In Milestone 1 of the TextMorph project, a secure user authentication system was developed using Streamlit, SQLite database, JWT (JSON Web Tokens), and Ngrok.

This module provides complete signup, login, dashboard access, and password recovery functionality with input validation and secure session handling. The system serves as the foundation for future TextMorph features such as text summarization and paraphrasing.

---Technologies Used

Python

Streamlit (Frontend UI)

SQLite Database

JWT Authentication

Ngrok (Public hosting)
